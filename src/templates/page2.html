API-Fatec

Para esse primeiro semestre de 2022 precisaremos produzir uma página web, ela será uma plataforma de terceirização de serviços para manutenção e resolução de problemas de hardware e/ou software. Para controlar as funções e informações no back-end usamos as tecnologias Python, Flask e MySQL, já para fazer todo o design e estilização no front-end foi utilizado HTML, CSS e JavaScript.
